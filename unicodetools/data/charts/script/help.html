About Script Charts

The Script charts provide an index to Unicode characters by script:

  • Characters with decompositions are listed under the script of the decomposed characters.
  • Characters with multiple script values are listed under each of those scripts.
  • Other characters (script values of Common or Inherited) are listed under General Category values at the end of the index.

Hovering over a cell will show the name of the character in that cell.

To properly view these charts, you need to have an updated browser and Unicode fonts that cover the characters you are interested in. For more information, see Display Problems.
